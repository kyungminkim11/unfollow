메일 앱 없이 바로 문의를 접수하세요

서비스 이용, 데이터 처리, Chrome 확장 프로그램, 오류 신고와 사업 제안을 한곳에서 접수합니다. 문의는 운영 데이터베이스에 저장되고 담당자 이메일로 즉시 알림이 전달됩니다.

문의 내용

답변받을 이메일을 정확히 입력해 주세요. 접수가 완료되면 화면에 접수번호가 표시됩니다.

문의 접수 시스템 운영 중

접수 후 처리

  • 문의 내용이 안전하게 저장됩니다.
  • 담당자 이메일로 신규 문의 알림이 전송됩니다.
  • 입력한 이메일을 기준으로 답변드립니다.
  • 접수번호로 문의 기록을 구분합니다.

오류 문의 전 확인

분석 화면 하단의 진단 파일 저장을 이용하면 사용자 아이디와 ZIP 원본 없이 브라우저 환경과 오류 상태를 전달할 수 있습니다.

ZIP 원본, Instagram 비밀번호, 인증 코드, 복구 코드는 보내지 마세요.

다른 빠른 경로
