맞팔체커 이용약관

본 약관은 라바랩스가 제공하는 웹 분석, Chrome Companion의 선택형 팔로우 취소 자동화, 문의와 뉴스레터 이용 조건을 정합니다.

시행일 2026.07.10최종 갱신 2026.07.15운영자 Lava Labs

제1조 목적

본 약관은 맞팔체커의 이용과 관련하여 서비스 운영자와 이용자 사이의 권리, 의무, 책임 및 기본적인 이용 조건을 정하는 것을 목적으로 합니다.

제2조 서비스 내용

  • Instagram JSON ZIP의 팔로워·팔로잉 관계 분석
  • 맞팔, 현재 비맞팔, 나를 팔로우 중인 계정 분류
  • 두 시점 ZIP 비교와 팔로워 이탈 대상 계산
  • 선택한 아이디 목록을 Chrome Companion으로 전달
  • 사용자가 시작하는 Instagram 화면 기반 팔로우 취소 배치 작업
  • 검색, CSV, 작업공간, 문의, 뉴스레터와 수요조사

웹은 Instagram 계정에 로그인하지 않습니다. Companion 자동화는 이용자가 Chrome에서 Instagram에 직접 로그인한 상태에서 사용자가 명시적으로 시작한 경우에만 작동합니다.

제3조 자동화 이용 조건

자동화는 선택 기능이며 분석만 사용할 수 있습니다. 목록을 전달한 것만으로 실행되지 않고, 이용자가 사이드패널에서 대상 수와 대기·휴식 설정을 확인하고 필수 확인란을 선택한 뒤 시작해야 합니다.

  • 한 번에 최대 30개까지 설정할 수 있습니다.
  • 로그인 화면, 보안 확인·챌린지 또는 연속 오류가 감지되면 중지합니다.
  • 이용자는 언제든 일시정지하거나 중지할 수 있습니다.
  • 사이드패널을 닫은 뒤 자동으로 다시 실행되지 않습니다.
  • Instagram 화면 구조나 정책 변경으로 기능이 제한되거나 중단될 수 있습니다.

제4조 이용자의 확인 책임

ZIP은 생성 시점의 스냅샷이므로 최신 상태와 다를 수 있습니다. Companion은 프로필의 현재 팔로우 상태를 다시 확인하지만, 이용자는 목록과 실행 설정이 자신의 의도와 일치하는지 시작 전에 확인해야 합니다. 서비스는 특정 계정을 언팔해야 한다고 판단하거나 권고하지 않습니다.

제5조 금지 행위

  • 본인이 관리할 권한이 없는 계정이나 타인의 ZIP을 이용하는 행위
  • 보안 확인, 접근 제한 또는 플랫폼의 기술적 보호조치를 우회하는 행위
  • 작업 상한·대기·중지 기능을 변조해 과도한 요청을 반복하는 행위
  • 계정 탈취, 스팸, 괴롭힘 또는 개인정보 무단 수집에 이용하는 행위
  • 문의·뉴스레터 폼에 허위 정보, 타인의 이메일 또는 자동화된 대량 요청을 입력하는 행위

제6조 계정 제한과 외부 플랫폼 위험

Instagram은 라바랩스가 운영하거나 통제하는 서비스가 아닙니다. 자동화된 반복 동작은 플랫폼이 비정상 활동으로 판단할 수 있으며 로그인 확인, 일시 제한 또는 기능 제한이 발생할 수 있습니다. 안전장치는 위험을 줄이기 위한 것이며 제한 발생이 없음을 보장하지 않습니다.

보안 확인이나 제한 화면이 나타나면 자동화를 즉시 중단하고 Instagram 화면의 안내를 직접 확인해야 합니다.

제7조 개인정보와 로컬 처리

ZIP 원본과 분석 결과는 브라우저에서 처리합니다. 자동화 대상으로 선택한 아이디와 작업 상태는 설치된 Chrome 확장 프로그램의 로컬 저장소에 보관되며 라바랩스 서버로 전송되지 않습니다. Companion은 비밀번호, 인증 코드, 쿠키, 세션 토큰과 Instagram 비공개 API를 사용하지 않습니다. 자세한 내용은 개인정보 처리방침과 데이터 처리 안내에서 확인할 수 있습니다.

제8조 서비스 변경·중단

운영자는 Instagram UI 변경, Chrome 정책 변경, 보안 문제, 장애 대응 또는 법적·운영상 필요가 있을 때 자동화 탐지 규칙, 지원 언어, 처리 상한 또는 기능 전체를 변경·중단할 수 있습니다.

제9조 문의, 뉴스레터와 수요조사

문의는 이용자가 제출한 범위에서 처리합니다. 뉴스레터와 프리미엄 수요조사는 선택 기능이며 회원가입이나 결제가 아닙니다. 각 페이지에서 수신 해지 또는 저장된 의견 삭제를 할 수 있습니다.

제10조 무료 베타와 유료 기능

현재 핵심 분석과 Companion은 무료 베타입니다. 향후 유료 상품을 판매하기 전에는 가격, 이용 기간, 자동 갱신, 청약철회와 환불 조건을 별도로 고지하고 동의를 받습니다.

제11조 지식재산권

서비스의 소프트웨어, 디자인, 문서, 브랜드와 운영 콘텐츠에 관한 권리는 운영자 또는 정당한 권리자에게 있습니다. 이용자가 자신의 ZIP과 분석 결과에 보유하는 권리는 본 약관으로 운영자에게 이전되지 않습니다.

제12조 책임의 범위

운영자는 안전한 기본 설정과 중지 기능을 제공하기 위해 노력합니다. 다만 이용자의 파일 선택, 대상 확인, 설정 변경, Instagram 상태·정책·UI 변화, 브라우저 환경과 통신 장애 등 운영자가 통제하기 어려운 사유로 발생한 문제에 대해서는 관련 법령이 허용하는 범위에서 책임이 제한될 수 있습니다.

제13조 외부 서비스와 비제휴 고지

맞팔체커와 Companion은 Instagram 또는 Meta가 공식 제공하거나 제휴한 서비스가 아닙니다. Instagram, Meta와 관련 명칭 및 상표는 각 권리자에게 귀속됩니다.

제14조 약관 변경과 문의

약관이 변경되는 경우 시행일과 주요 내용을 서비스 화면에 안내합니다. 이용 문의는 맞팔체커 문의 페이지에서 접수할 수 있습니다.

자동화 실행 전 목록과 설정을 반드시 확인하고, Instagram에서 보안 확인이나 제한 안내가 나타나면 즉시 중지하세요.
