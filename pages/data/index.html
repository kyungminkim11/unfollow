ZIP 분석과 선택 제출 정보는 서로 분리됩니다

핵심 분석은 브라우저 안에서 진행됩니다. 문의, 뉴스레터 또는 프리미엄 수요조사에 직접 참여한 경우에만 해당 정보가 서버에 저장되며, ZIP 원본이나 분석 결과가 함께 전송되지는 않습니다.

ZIP 원본 서버 전송 없음분석 결과 로컬 처리선택 제출 정보만 분리 저장

ZIP 분석 흐름

1사용자가 ZIP을 선택합니다Instagram에서 직접 내려받은 JSON ZIP을 현재 기기에서 선택합니다.
2브라우저가 파일을 읽습니다팔로워·팔로잉 JSON을 브라우저 메모리에서 읽고 관계를 분류합니다.
3결과를 화면에 표시합니다분석 결과는 서버가 아니라 현재 브라우저 화면과 로컬 저장소에서 사용됩니다.

서버로 보내지 않는 정보

  • Instagram ZIP 원본
  • ZIP 내부 JSON 원문
  • 분석된 사용자 아이디 전체 목록
  • 맞팔·언팔 검토 결과 전체
  • Instagram 로그인 정보와 비밀번호

브라우저에 저장될 수 있는 정보

  • 작업공간 이름
  • 검토 완료 상태와 메모
  • 화면 설정과 마지막 사용 상태
  • 작업공간 연결용 비식별 해시 스케치

작업공간 관리나 브라우저 사이트 데이터 삭제 기능으로 제거할 수 있습니다.

문의 접수 정보

  • 이름과 답변 이메일
  • 문의 유형과 제목
  • 문의 내용
  • 개인정보 동의 기록
  • 접수·처리·이메일 알림 상태

뉴스레터 신청 정보

  • 이메일 주소
  • 개인정보·이메일 수신 동의
  • 동의 버전과 신청 일시
  • 활성 상태
  • 임의 수신 해지 토큰

프리미엄 수요조사 정보

  • 이메일 주소
  • 관심 기능
  • 희망 가격대
  • 관리 계정 수 범위
  • 추가 의견과 동의 기록

문의와 이메일 알림 흐름

1이용자가 문의를 제출합니다서버가 필수 입력, 이메일 형식, 동의와 자동 제출 방지 항목을 검증합니다.
2접수번호와 기록을 저장합니다문의는 ZIP 분석 정보와 분리된 업무 테이블에 저장됩니다.
3운영자에게 알립니다운영자 이메일 알림을 전송하고 관리자 화면에서 처리 상태를 관리합니다.

뉴스레터 캠페인 흐름

1관리자가 캠페인을 작성합니다제목, 본문, 버튼과 발송 대상을 관리자 전용 화면에서 저장합니다.
2대기열에서 수신자를 확정합니다활성 구독자의 이메일과 수신 해지 토큰을 캠페인별 전달 목록으로 저장합니다.
3발송 상태를 기록합니다대기·성공·실패·취소 상태를 저장해 중복 발송을 줄이고 실패 건을 재시도합니다.

이메일 제공자에게 전달되는 정보

  • 수신 이메일 주소
  • 발신자와 회신 주소
  • 이메일 제목과 본문
  • 개별 수신 해지 링크
  • 접수 또는 캠페인 구분 메타데이터

운영자 알림에는 Formspree가 사용될 수 있으며, 접수 확인과 뉴스레터 발송 기능이 연결된 경우 Resend로 전달됩니다.

발송 대기열에 포함되지 않는 정보

  • Instagram ZIP 원본
  • 분석된 계정 목록
  • 작업공간 메모
  • 프리미엄 설문의 자유 의견
  • 카드 또는 결제 정보

저장 위치와 접근

문의, 뉴스레터, 수요조사, 캠페인과 전달 목록은 Supabase 기반 데이터베이스에 분리해 저장되며 현재 데이터베이스 리전은 서울입니다. 공개 페이지에서는 데이터베이스를 직접 읽을 수 없습니다. 관리자 API는 유효한 로그인 토큰과 허용 이메일을 모두 확인합니다.

분석 기능은 문의, 뉴스레터나 수요조사에 참여하지 않아도 사용할 수 있습니다. 선택 제출 여부는 ZIP 분석 결과에 영향을 주지 않습니다.

삭제와 해지

뉴스레터는 수신 해지 페이지에서 이메일을 입력하거나, 발송 메일의 개별 해지 링크를 열어 삭제할 수 있습니다. 프리미엄 의견은 수요조사 화면에서 삭제할 수 있습니다. 캠페인 초안을 삭제하면 연결된 전달 목록도 함께 삭제됩니다. 문의 기록이나 추가 요청은 문의 페이지에서 접수할 수 있습니다.

앞으로 회원가입이나 결제가 생기면?

회원가입, 클라우드 동기화 또는 결제를 도입하는 경우 실제 수집 항목, 저장 위치, 보유 기간, 가격과 환불 조건을 기능 활성화 전에 다시 고지합니다. 현재 뉴스레터 신청이나 수요조사 참여는 회원가입 또는 유료 구독으로 자동 전환되지 않습니다.
