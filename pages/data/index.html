ZIP 분석과 선택 신청 정보는 서로 분리됩니다

맞팔체커의 핵심 분석은 브라우저 안에서 진행됩니다. 뉴스레터나 프리미엄 수요조사에 직접 참여한 경우에만 해당 신청 정보가 서버에 저장되며, ZIP 원본이나 분석 결과가 함께 전송되지는 않습니다.

ZIP 원본 서버 전송 없음분석 결과 로컬 처리선택 신청 정보만 분리 저장

ZIP 분석 흐름

1사용자가 ZIP을 선택합니다Instagram에서 직접 내려받은 JSON ZIP을 현재 기기에서 선택합니다.
2브라우저가 파일을 읽습니다팔로워·팔로잉 JSON을 브라우저 메모리에서 읽고 관계를 분류합니다.
3결과를 화면에 표시합니다분석 결과는 서버가 아니라 현재 브라우저 화면과 로컬 저장소에서 사용됩니다.

서버로 보내지 않는 정보

  • Instagram ZIP 원본
  • ZIP 내부 팔로워·팔로잉 JSON 원문
  • 분석된 사용자 아이디 전체 목록
  • 맞팔·언팔 검토 결과 전체
  • Instagram 로그인 정보와 비밀번호

브라우저에 저장될 수 있는 정보

  • 작업공간 이름
  • 검토 완료 상태와 메모
  • 화면 설정과 마지막 사용 상태
  • 개인정보를 직접 포함하지 않는 작업공간 연결용 해시 스케치

브라우저의 사이트 데이터 삭제 기능이나 작업공간 관리에서 지울 수 있습니다.

뉴스레터 신청 시 저장되는 정보

  • 이메일 주소
  • 개인정보 수집·이용 동의 상태
  • 이메일 수신 동의 상태
  • 동의 버전과 신청·해지 일시
  • 활성 또는 해지 상태

프리미엄 수요조사 시 저장되는 정보

  • 이메일 주소
  • 관심 기능
  • 희망 가격대
  • 관리 계정 수 범위
  • 추가 의견과 개인정보 동의 기록

뉴스레터 저장 목적

프리미엄 출시, 중요한 기능 변경, 무료 베타 관련 공지를 이메일로 전달하고 신청·해지 상태를 관리하기 위한 목적에만 사용합니다.

회원가입 정보, 결제 정보, ZIP 분석 데이터와 결합하지 않습니다.

수요조사 저장 목적

많이 요청된 기능과 가격 범위를 집계해 개발 우선순위를 정하고, 같은 이메일의 의견을 최신 내용으로 갱신하거나 삭제하기 위해 사용합니다.

수요조사 참여만으로 뉴스레터가 발송되지는 않습니다.

어디에 저장되나요?

뉴스레터와 프리미엄 수요조사 정보는 Supabase 기반 데이터베이스에 분리해 저장됩니다. 현재 프로젝트의 데이터베이스 리전은 서울로 설정되어 있습니다. 서비스 운영과 보안 지원을 위해 인프라 제공업체가 관련 시스템을 처리할 수 있으며, 처리 방식이 변경되면 개인정보 처리방침을 갱신합니다.

분석 기능은 뉴스레터나 수요조사에 참여하지 않아도 사용할 수 있습니다. 선택 신청 여부는 ZIP 분석 결과에 영향을 주지 않습니다.

관리자 접근

구독자와 수요조사 정보는 등록된 라바랩스 관리자 이메일로 일회용 로그인 링크를 받은 뒤에만 확인할 수 있습니다. 공개 웹페이지는 데이터베이스를 직접 읽을 수 없으며, 관리자 API가 로그인 토큰과 허용 이메일을 확인합니다. 구독자 CSV 내보내기와 삭제 같은 관리 작업은 별도 기록으로 남깁니다.

삭제와 해지

뉴스레터는 수신 해지 페이지에서 이메일을 입력해 해지할 수 있습니다. 프리미엄 의견은 수요조사 화면에서 같은 이메일을 입력한 뒤 삭제할 수 있습니다. 브라우저 작업 기록은 맞팔체커의 작업공간 관리 또는 브라우저 사이트 데이터 삭제 기능으로 제거할 수 있습니다. 추가 삭제 요청은 이메일 문의로 접수할 수 있습니다.

앞으로 회원가입이 생기면?

유료 기능을 위해 회원가입, 클라우드 동기화, 결제가 도입되는 경우에는 수집 항목, 저장 위치, 보유 기간, 결제·환불 조건을 기능 활성화 전에 다시 고지하고 필요한 동의를 받습니다. 현재 뉴스레터 신청이나 수요조사 참여가 자동으로 회원가입 또는 유료 구독으로 전환되지는 않습니다.
