문제 해결

스캔이나 분석이 되지 않을 때 확인하세요

확장 프로그램 버전, Instagram 로그인 상태, 내 프로필 여부, 목록 로딩과 ZIP 형식을 순서대로 확인하면 대부분의 문제를 찾을 수 있습니다.

오류 상황별 해결 방법

상황확인 방법
Companion이 연결되지 않음기존 v22 폴더가 설치돼 있다면 제거하거나 v23 폴더로 다시 불러온 뒤 맞팔체커를 새로고침하세요.
내 프로필을 확인하지 못함Instagram에서 로그인한 계정의 내 프로필을 직접 연 뒤 아이디를 다시 감지하세요. 다른 사람의 프로필은 스캔하지 않습니다.
팔로워·팔로잉 버튼을 찾지 못함프로필 페이지가 완전히 로드됐는지 확인하고 새로고침하세요. Instagram 화면 구조가 바뀐 경우 문의로 오류 문구를 알려 주세요.
목록 창을 열지 못함팝업이나 목록 창이 이미 열려 있으면 닫고 다시 시작하세요. 탭 확대율을 기본값으로 되돌리는 것도 도움이 됩니다.
일부 목록일 수 있음 경고Instagram이 목록 추가 로딩을 멈췄거나 네트워크가 느린 상태입니다. 잠시 뒤 다시 스캔하거나 공식 JSON ZIP으로 분석하세요.
보안 확인·챌린지 감지스캔과 작업은 자동 중지됩니다. Instagram 화면에서 확인 절차를 직접 완료하고 충분히 쉬었다가 다시 시작하세요.
followers_*.json 또는 following.json을 찾지 못함공식 데이터 요청 시 파일 형식을 JSON으로 설정하고 팔로워 및 팔로잉을 포함해 다시 요청하세요.
연속 오류로 작업 중지실패 항목을 즉시 반복하지 말고 현재 화면 언어와 버튼 상태를 확인하세요.

팔로워 수와 수집 수가 달라요

Instagram의 표시 숫자가 반올림되거나 비활성·삭제 계정, 로딩 상태 때문에 실제 화면 목록과 차이가 날 수 있습니다. 차이가 크면 일부 목록 경고를 확인하고 다시 스캔하세요.

스캔 중 다른 탭을 사용해도 되나요?

다른 탭을 보는 것은 가능하지만, 스캔에 사용하는 Instagram 탭의 프로필과 목록 창을 닫거나 다른 주소로 이동하면 수집이 중지됩니다.

맞팔 아닌 명단은 어떻게 계산하나요?

현재 팔로잉 명단에서 현재 팔로워 명단을 뺍니다. 따라서 나는 팔로우하지만 상대는 나를 팔로우하지 않는 계정만 포함됩니다.

현재 스캔으로 과거 언팔을 알 수 있나요?

아닙니다. 한 시점의 명단으로는 과거 변화를 알 수 없습니다. 이전·최신 공식 ZIP을 비교하거나 서로 다른 날짜의 기록이 필요합니다.

결과는 어디에 저장되나요?

팔로워·팔로잉·맞팔 아닌 명단과 작업 상태는 설치된 확장 프로그램의 로컬 저장소에 남습니다. 라바랩스 서버나 다른 기기로 자동 전송되지 않습니다.

목록을 바로 실행하나요?

아닙니다. 스캔이 끝나도 자동 실행되지 않습니다. 맞팔 아닌 명단을 검토해 작업 목록을 만든 뒤 별도 확인란과 시작 버튼을 눌러야 합니다.

자주 묻는 질문

팔로워·팔로잉을 Instagram 비공개 API로 가져오나요?

아니요. Companion은 내 프로필에서 사용자가 볼 수 있는 목록 창의 프로필 링크를 읽고 스크롤합니다. 비공개 API 요청을 만들지 않습니다.

비밀번호나 로그인 쿠키를 읽나요?

아니요. 비밀번호, 인증 코드, 쿠키와 세션 토큰을 읽지 않습니다. 이미 로그인된 Instagram 탭의 화면만 사용합니다.

스캔 결과를 CSV로 저장할 수 있나요?

네. Companion에서 팔로워, 팔로잉 또는 맞팔 아닌 탭을 선택하고 CSV 버튼을 누르면 현재 명단을 저장할 수 있습니다.

사이드패널을 닫으면 어떻게 되나요?

진행 중인 스캔이나 팔로우 취소 작업은 중지됩니다. 저장된 명단과 완료 상태는 남지만 자동 재시작하지 않습니다.

기존 ZIP 분석은 없어졌나요?

아닙니다. ZIP 방식은 대량 분석과 과거 A·B 시점 비교에 계속 사용할 수 있습니다.

계속 해결되지 않는 경우

Companion 버전, 화면에 표시된 오류 문구, Chrome 버전과 Instagram 화면 언어를 보내 주세요. ZIP 원본, 비밀번호, 인증 코드 또는 쿠키는 보내지 마세요.
