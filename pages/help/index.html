문제 해결

분석이 되지 않을 때 확인하세요

대부분의 오류는 다운로드 형식이 HTML이거나, 팔로워·팔로잉 파일이 ZIP에 포함되지 않았거나, 다운로드가 완료되지 않은 경우에 발생합니다.

오류 메시지별 해결 방법

상황확인 방법
followers_*.json 또는 following.json을 찾지 못함Instagram 데이터 요청 시 파일 형식을 JSON으로 설정하고 ‘팔로워 및 팔로잉’을 포함해 다시 요청하세요.
ZIP 파일이 손상됨다운로드가 완전히 끝났는지 확인한 뒤 파일을 다시 내려받으세요. 메신저로 전달하며 파일이 변형된 경우 원본을 사용하세요.
파일이 80MB를 초과함전체 Instagram 정보 대신 팔로워 및 팔로잉 항목만 선택해 새 ZIP을 받으세요.
압축 해제를 지원하지 않는 브라우저최신 Chrome, Edge 또는 Safari에서 다시 열어 주세요.
결과가 현재 Instagram과 다름결과는 ZIP 생성 시점의 스냅샷입니다. 실제 변경 전 해당 프로필을 직접 확인하세요.

작업 기록은 어디에 저장되나요?

검토 상태와 작업공간 이름은 현재 브라우저의 로컬 저장소에 저장됩니다. 다른 기기나 다른 브라우저에는 자동으로 동기화되지 않습니다.

자동으로 언팔하나요?

아닙니다. 맞팔체커는 관계를 분류하고 프로필 링크를 제공할 뿐입니다. 팔로우 변경은 사용자가 Instagram에서 직접 진행합니다.

두 ZIP 비교는 어떻게 하나요?

분석 화면의 ‘관계 변화 비교’에서 이전 ZIP과 최신 ZIP을 각각 선택하세요. 새 팔로워, 팔로워 이탈, 새 맞팔 등의 변화를 확인할 수 있습니다.

문의할 때 무엇을 보내야 하나요?

분석 화면 하단의 ‘진단 파일 저장’을 사용하면 사용자 아이디와 ZIP 파일명 없이 브라우저 지원 정보와 오류 상태를 저장할 수 있습니다.

자주 묻는 질문

ZIP 파일을 서버에 올리나요?

아니요. 파일 선택과 분석은 현재 브라우저 안에서 수행되며 ZIP 내용은 외부 서버로 전송되지 않습니다.

뉴스레터를 신청하면 ZIP 정보도 저장되나요?

아니요. 뉴스레터에는 이메일과 동의 기록만 저장되며, ZIP 원본과 분석 결과는 함께 전송되지 않습니다.

뉴스레터 신청이 유료 구독인가요?

아닙니다. 회원가입이나 결제가 아니며 프리미엄 출시와 주요 업데이트 소식을 받기 위한 선택 신청입니다.

뉴스레터를 해지할 수 있나요?

수신 해지 페이지에서 등록 이메일을 입력하면 해지와 저장 정보 삭제를 요청할 수 있습니다.

비공개 계정도 표시되나요?

Instagram 다운로드 데이터에 사용자 아이디가 포함되어 있다면 관계 목록에 표시될 수 있습니다. 프로필의 현재 공개 상태를 별도로 조회하지는 않습니다.

결과 목록에서 바로 프로필을 열어도 안전한가요?

프로필 버튼을 누르면 Instagram 웹사이트가 새 탭으로 열립니다. 이때부터는 Instagram의 로그인 상태와 정책이 적용됩니다.

계속 해결되지 않는 경우

진단 파일과 화면에 표시된 오류 문구를 함께 보내 주세요. ZIP 원본이나 사용자 아이디 목록은 보내지 않아도 됩니다.
