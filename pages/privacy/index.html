2. Instagram ZIP과 분석 결과 처리

사용자가 선택한 ZIP은 브라우저 메모리에서 읽습니다. 팔로워·팔로잉 JSON 원문, 사용자 아이디 목록, 관계 분석 결과를 맞팔체커 서버, 뉴스레터 데이터베이스 또는 프리미엄 수요조사 데이터베이스로 전송하지 않습니다.

  • Instagram 아이디·비밀번호 또는 로그인 토큰을 요구하지 않습니다.
  • 자동 팔로우·언팔 권한을 받지 않습니다.
  • 결과는 ZIP 생성 시점의 스냅샷이며 현재 상태와 다를 수 있습니다.

자세한 흐름은 데이터 처리 안내에서 확인할 수 있습니다.

3. 뉴스레터 신청과 동의

뉴스레터는 서비스 이용에 필수인 회원가입 절차가 아니라 희망자가 별도로 신청하는 선택 기능입니다. 신청 화면에서 개인정보 수집·이용과 업데이트·프리미엄 출시 이메일 수신에 각각 동의한 경우에만 등록됩니다.

  • 현재 카드 번호나 결제 정보는 수집하지 않습니다.
  • 뉴스레터 신청이 회원가입 또는 유료 구독으로 자동 전환되지 않습니다.
  • ZIP 분석 여부와 뉴스레터 신청 여부를 결합하지 않습니다.
  • 신청 정보를 제3자의 일반 광고 목록으로 판매하거나 제공하지 않습니다.

4. 프리미엄 수요조사

프리미엄 페이지에서 이용자가 직접 참여한 경우에만 이메일과 선택한 의견을 저장합니다. 뉴스레터 수신 동의와는 분리되며, 수요조사에 참여해도 이메일 광고가 자동 발송되지 않습니다.

  • 같은 이메일로 다시 제출하면 기존 의견을 최신 내용으로 갱신합니다.
  • 이메일은 중복 응답을 줄이고 이용자가 자신의 의견을 갱신·삭제하는 데 사용합니다.
  • 선택 기능과 희망 가격은 통계로 집계해 개발 우선순위를 판단하는 데 사용합니다.
  • 추가 의견에는 민감한 개인정보나 타인의 정보를 입력하지 않도록 안내합니다.

5. 보유 기간과 삭제

뉴스레터

수신 해지 페이지에서 등록 이메일을 입력하면 발송 대상에서 제외하고, 해당 이메일 및 연결된 동의 기록을 삭제합니다.

프리미엄 수요조사

수요조사 화면에서 이메일을 입력하고 ‘저장된 내 의견 삭제’를 선택하면 해당 이메일로 저장된 설문 정보가 삭제됩니다.

브라우저 작업 기록

작업공간 관리에서 개별·전체 삭제하거나 브라우저 설정에서 unfollow.lavalabs.co.kr의 사이트 데이터를 삭제할 수 있습니다. ZIP 원본은 로컬 저장소에 저장하지 않습니다.

별도 삭제 요청은 개인정보 문의 이메일로 접수할 수 있습니다.

7. 개인정보의 제3자 제공

라바랩스는 이용자의 개인정보를 판매하지 않으며, 원칙적으로 제3자에게 제공하지 않습니다. 이용자의 별도 동의가 있거나 법령에 근거한 요청이 있는 경우에만 필요한 범위에서 처리합니다.

8. 자동 저장 기술과 로컬 저장소

맞팔체커는 일반 이용자에게 로그인 쿠키를 요구하지 않고 브라우저의 로컬 저장소와 서비스 워커 캐시를 사용해 작업 상태, 화면 설정, 오프라인 사용에 필요한 정적 파일을 저장할 수 있습니다. 이 정보는 이용자의 기기에 남으며 브라우저 설정에서 삭제할 수 있습니다.

관리자 화면은 일회용 이메일 링크로 로그인하며, 관리자 인증 토큰은 해당 브라우저 탭의 세션 저장소에만 보관되도록 설계했습니다.

이용자는 본인의 뉴스레터 및 프리미엄 수요조사 정보에 대해 열람, 정정, 삭제, 처리 정지를 요청할 수 있습니다. 온라인 해지·삭제 기능을 이용하거나 lavalabs.ceo@gmail.com으로 요청할 수 있습니다. 본인 확인이 필요한 경우 최소한의 확인 절차를 요청할 수 있습니다.
