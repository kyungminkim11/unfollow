ZIP 내용은 외부 서버로 전송되지 않습니다

맞팔체커는 정적 웹앱으로 동작합니다. 사용자가 선택한 Instagram ZIP은 브라우저 메모리에서 읽고 분류하며, 분석을 위해 라바랩스 서버로 업로드하지 않습니다.

서비스로 돌아가기도움말 보기

브라우저에서 처리하는 정보

  • 선택한 ZIP 내부의 팔로워·팔로잉 JSON
  • 분류된 사용자 아이디와 관계 상태
  • 두 ZIP 사이의 관계 변화 결과

이 정보는 분석 중 브라우저 메모리에서 사용됩니다.

브라우저에 저장되는 정보

  • 계정 검토 상태
  • 작업공간 이름과 마지막 수정 시각
  • 화면 설정과 일부 사용 환경

로컬 저장소 정보는 같은 브라우저에서만 사용할 수 있습니다.

수집하지 않는 정보

  • Instagram 비밀번호 또는 로그인 토큰
  • ZIP 원본과 ZIP 내부 JSON 원문
  • 분석된 사용자 아이디 목록의 서버 저장
  • 자동 팔로우·자동 언팔 권한

외부 사이트 이동

결과에서 프로필을 열거나 Instagram 계정 센터 링크를 선택하면 Instagram 또는 Meta 사이트로 이동합니다. 이동한 뒤의 데이터 처리는 해당 서비스의 정책이 적용됩니다.

저장 정보 삭제 방법

  1. 맞팔체커의 작업공간 관리를 엽니다분석 화면의 ‘작업 기록’ 또는 하단 ‘작업공간 관리’를 선택합니다.
  2. 개별 작업공간을 삭제합니다삭제할 작업공간의 ‘삭제’를 선택하면 해당 브라우저에 저장된 진행 기록이 제거됩니다.
  3. 브라우저 데이터 전체 삭제도 가능합니다브라우저 설정에서 unfollow.lavalabs.co.kr의 사이트 데이터 또는 로컬 저장소를 삭제할 수 있습니다.
ZIP 원본은 맞팔체커의 로컬 저장소에 보관하지 않습니다. 파일 선택을 취소하거나 페이지를 닫으면 브라우저 메모리의 분석 데이터는 유지되지 않을 수 있습니다.

진단 파일

‘진단 파일 저장’ 기능은 브라우저 종류, 화면 크기, 지원 기능, 작업공간 개수, 화면에 표시된 집계 수치처럼 문제 해결에 필요한 기술 정보를 저장합니다. 사용자 아이디, ZIP 파일명, ZIP 내용과 작업 메모는 포함하지 않도록 설계되어 있습니다.
