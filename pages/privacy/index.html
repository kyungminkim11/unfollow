2. Instagram 공식 ZIP

선택한 ZIP은 현재 웹 브라우저 메모리에서 읽습니다. JSON 원문, 사용자 아이디 전체 목록과 관계 분석 결과를 서버, 문의 데이터베이스 또는 뉴스레터 데이터베이스로 전송하지 않습니다.

  • Instagram 아이디·비밀번호, 인증 코드 또는 로그인 토큰을 요구하지 않습니다.
  • ZIP 결과는 생성 시점의 기록이며 현재 상태와 다를 수 있습니다.
  • A·B 비교는 두 ZIP의 관계 목록을 현재 브라우저에서 계산합니다.

3. Chrome Companion 관계 스캔

이용자가 사이드패널에서 스캔을 시작한 경우에만 로그인한 계정의 내 프로필에서 팔로워와 팔로잉 목록 창을 열고 화면에 표시되는 프로필 링크를 수집합니다.

  • 내 프로필 경로, 팔로워·팔로잉 링크, 목록 창의 프로필 링크와 스크롤 상태를 확인합니다.
  • 팔로워와 팔로잉의 차집합으로 맞팔 아닌 명단을 확장 프로그램 안에서 계산합니다.
  • 수집한 아이디와 결과는 chrome.storage.local에 보관하며 라바랩스 서버로 전송하지 않습니다.
  • 비밀번호, 인증 코드, 쿠키, 세션 토큰, DM, 게시물 내용과 Instagram 비공개 API를 읽거나 호출하지 않습니다.
  • 탭 또는 목록 창이 닫히거나 로그인·보안 확인 화면이 감지되면 중지합니다.

4. Companion 팔로우 취소 작업

스캔 또는 ZIP 분석 후 이용자가 직접 선택하고 확인한 아이디만 작업 목록에 포함됩니다. 목록 생성만으로 실행되지 않으며 사이드패널에서 다시 확인하고 시작해야 합니다.

  • 현재 프로필 경로와 화면에 표시된 팔로잉·팔로우 취소 버튼을 확인합니다.
  • 처리 결과, 건너뜀, 실패와 실행 설정은 확장 프로그램 로컬 저장소에 남습니다.
  • 사이드패널을 닫거나 중지하면 자동으로 다시 시작하지 않습니다.

5. 서비스 문의

문의 페이지에서 이용자가 직접 제출한 정보만 서버에 저장합니다. 접수번호를 발급하고 담당자 이메일로 알림을 보냅니다. ZIP 원본, 관계 스캔 명단, 비밀번호와 인증 코드는 문의 내용에 포함하지 않도록 안내합니다.

6. 뉴스레터와 프리미엄 수요조사

뉴스레터와 수요조사는 선택 기능이며 관계 분석·스캔과 분리됩니다. 뉴스레터 신청은 회원가입이나 결제가 아니며 온라인 해지 기능을 제공합니다. 스캔 명단과 작업 상태는 뉴스레터 또는 수요조사 정보와 결합하지 않습니다.
