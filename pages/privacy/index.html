2. Instagram ZIP과 분석 결과

사용자가 선택한 ZIP은 브라우저 메모리에서 읽습니다. 팔로워·팔로잉 JSON 원문, 사용자 아이디 전체 목록과 관계 분석 결과를 맞팔체커 서버, 뉴스레터 데이터베이스 또는 수요조사 데이터베이스로 전송하지 않습니다.

  • Instagram 아이디·비밀번호 또는 로그인 토큰을 요구하지 않습니다.
  • 자동 팔로우·언팔 권한을 받지 않습니다.
  • 결과는 ZIP 생성 시점의 스냅샷이며 현재 상태와 다를 수 있습니다.

3. 뉴스레터 신청과 수신 해지

뉴스레터는 회원가입이나 결제가 아니라 희망자가 별도로 신청하는 선택 기능입니다. 개인정보 수집·이용과 이메일 수신에 동의한 경우에만 등록됩니다.

  • 구독자마다 임의 UUID 수신 해지 토큰을 발급합니다.
  • 메일의 해지 링크에는 이메일 주소 대신 토큰만 포함합니다.
  • 해지 링크 또는 이메일 입력 방식으로 구독 정보와 연결 동의 기록을 삭제할 수 있습니다.
  • 뉴스레터 신청이 회원가입, 유료 구독 또는 프리미엄 수요조사 참여로 자동 전환되지 않습니다.

4. 프리미엄 수요조사

프리미엄 페이지에서 이용자가 직접 참여한 경우에만 이메일과 선택 의견을 저장합니다. 뉴스레터 동의와 분리되며, 수요조사 참여만으로 이메일이 발송되지는 않습니다.

  • 같은 이메일로 다시 제출하면 기존 의견을 갱신합니다.
  • 온라인에서 저장된 의견을 직접 삭제할 수 있습니다.
  • 추가 의견에는 민감한 정보나 타인의 정보를 입력하지 않도록 안내합니다.

5. 캠페인 대기열과 발송 기록

관리자가 캠페인을 발송 대기열에 넣으면 그 시점의 활성 구독자 이메일, 구독자 식별자와 수신 해지 토큰을 캠페인별 전달 목록으로 저장합니다. 이는 발송 도중 구독 상태가 바뀌는 경우를 관리하고 중복 발송을 방지하기 위한 스냅샷입니다.

  • 전달 목록에는 Instagram ZIP이나 분석 결과가 포함되지 않습니다.
  • 발송 상태는 대기, 발송 중, 성공, 실패, 건너뜀으로 기록할 수 있습니다.
  • 관리자가 캠페인을 삭제하면 연결된 전달 목록도 함께 삭제됩니다.
  • 실패 원인은 운영에 필요한 범위에서만 저장하며 민감한 내용을 기록하지 않도록 제한합니다.

7. 제3자 제공과 판매 금지

라바랩스는 개인정보를 판매하지 않으며 일반 광고 목록으로 제공하지 않습니다. 이용자의 별도 동의, 서비스 제공에 필요한 처리 위탁 또는 법령에 근거한 요청이 있는 경우에만 필요한 범위에서 처리합니다.

9. 자동 저장 기술과 관리자 인증

일반 이용자에게 로그인 쿠키를 요구하지 않으며 로컬 저장소와 서비스 워커 캐시를 사용해 작업 상태, 화면 설정과 정적 파일을 저장할 수 있습니다. 관리자 화면은 이메일 매직링크를 사용하며 인증 토큰은 해당 브라우저 탭의 세션 저장소에 보관합니다. 관리자 페이지는 검색 노출과 오프라인 캐시에서 제외합니다.
