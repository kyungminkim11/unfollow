분석부터 팔로우 취소 작업까지

Instagram 공식 JSON ZIP을 준비하고, 현재 비맞팔 또는 A·B 시점의 팔로워 이탈 대상을 계산한 뒤 필요한 계정만 Companion으로 보냅니다.

ZIP 분석 시작Instagram 계정 센터 열기

1. Instagram 데이터 요청

  1. Instagram 계정 센터를 엽니다설정에서 ‘내 정보 및 권한’과 ‘내 정보 다운로드’로 이동합니다.
  2. ‘팔로워 및 팔로잉’을 포함합니다전체 정보가 아니라 필요한 관계 데이터만 선택하면 파일 크기를 줄일 수 있습니다.
  3. 파일 형식을 JSON으로 설정합니다HTML이 아닌 JSON이어야 맞팔체커가 분석할 수 있습니다.
  4. 완료된 ZIP을 그대로 저장합니다압축을 풀거나 내부 파일을 수정하지 마세요.

2-A. 현재 비맞팔 찾기

최신 ZIP 한 개를 기본 분석 화면에 선택합니다. 현재 비맞팔은 최신 시점에서 나는 팔로우하지만 상대는 나를 팔로우하지 않는 계정입니다.

2-B. 나를 언팔한 계정 찾기

관계 변화 비교에 이전 ZIP과 최신 ZIP을 선택합니다. 자동화 대상은 단순 팔로워 이탈 전체가 아니라, 이탈한 사람 중 최신 시점에도 내가 팔로우 중인 계정만 계산합니다.

3. Companion 설치

  1. 웹의 ‘Companion 받기’를 누릅니다matchal-companion-v22.zip 파일이 내려받아집니다.
  2. ZIP을 폴더에 압축 해제합니다압축 파일 자체가 아니라 풀린 폴더를 Chrome에 등록합니다.
  3. chrome://extensions를 엽니다개발자 모드를 켜고 ‘압축해제된 확장 프로그램을 로드합니다’를 눌러 폴더를 선택합니다.
  4. 맞팔체커 페이지를 새로고침합니다‘확장 프로그램 연결됨’이 표시되면 준비가 끝납니다.

4. 목록 선택과 전달

  1. 현재 비맞팔 또는 팔로워 이탈 대상을 선택합니다목록에서 제외할 계정은 체크를 해제할 수 있습니다.
  2. 자동 클릭 작업임을 확인합니다확인란을 선택하기 전에는 전송 버튼이 활성화되지 않습니다.
  3. 사이드패널로 목록을 보냅니다ZIP 원본이 아니라 선택한 Instagram 아이디만 확장 프로그램 로컬 저장소로 전달됩니다.

5. 처리 수와 안전 간격 설정

설정기본값·범위의미
이번 처리 수기본 10개 · 최대 30개한 번의 시작 버튼으로 처리할 최대 계정 수
계정 사이 대기기본 8~15초 · 최소 5초각 프로필 사이에 임의로 기다리는 시간
정기 휴식기본 10건마다 60초연속 작업 사이에 긴 휴식을 넣는 설정
안전 중지로그인·챌린지·연속 오류확인 화면이나 2회 연속 버튼 탐지 실패 시 자동 중지

실행 중 조작

  • 일시정지 후 같은 위치에서 계속
  • 즉시 중지 후 처리 중 항목을 대기로 복원
  • 실패 항목만 다시 대기로 변경
  • 이미 언팔된 계정은 건너뜀 처리

중요한 확인

  • Instagram에 직접 로그인된 Chrome 탭 필요
  • ZIP은 생성 시점의 스냅샷
  • 프로필 현재 상태는 실행 시 다시 확인
  • 비밀번호·쿠키·인증 코드는 읽지 않음
