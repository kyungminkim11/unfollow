Instagram 데이터 ZIP을 준비하는 방법

맞팔체커는 Instagram이 제공하는 데이터 다운로드 ZIP을 사용합니다. 로그인 정보는 입력하지 않으며, ZIP은 압축을 풀지 않은 상태로 선택하면 됩니다.

ZIP 분석 시작Instagram 계정 센터 열기

데이터 요청 순서

  1. Instagram의 계정 센터를 엽니다Instagram 앱의 설정 메뉴 또는 웹 계정 센터에서 ‘내 정보 및 권한’ 항목으로 이동합니다.
  2. 내 정보 다운로드를 선택합니다다운로드할 Instagram 계정을 선택하고, 전체 정보 대신 필요한 정보 유형을 선택합니다.
  3. ‘팔로워 및 팔로잉’을 포함합니다맞팔 분석에 필요한 팔로워·팔로잉 관계 데이터가 포함되도록 선택합니다.
  4. 파일 형식을 JSON으로 설정합니다HTML이 아니라 JSON을 선택해야 분석할 수 있습니다. 날짜 범위는 확인하려는 기간에 맞춰 선택하세요.
  5. 완료된 ZIP을 내려받습니다Instagram에서 준비 완료 알림을 받으면 ZIP을 기기에 저장합니다. 압축을 풀거나 파일명을 바꿀 필요는 없습니다.
  6. 맞팔체커에서 ZIP을 선택합니다분석 화면에서 ZIP을 선택하면 브라우저가 파일 내부의 followers_*.json과 following.json을 확인합니다.
Instagram의 메뉴 명칭과 위치는 앱 버전이나 계정 센터 구성에 따라 달라질 수 있습니다. 핵심은 팔로워 및 팔로잉 데이터와 JSON 형식입니다.

선택해야 하는 설정

  • 정보 유형: 팔로워 및 팔로잉
  • 파일 형식: JSON
  • 파일: ZIP 상태 유지
  • 권장 브라우저: 최신 Chrome, Edge, Safari

분석 전에 확인할 점

  • ZIP 확장자가 맞는지 확인
  • 파일 크기가 80MB 이하인지 확인
  • 다운로드 완료 전 임시 파일이 아닌지 확인
  • 결과는 ZIP 생성 시점의 상태임을 확인
