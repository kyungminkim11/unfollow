웹 스캔 열기
두 가지 사용 방식

현재 팔로워·팔로잉은 다운로드 없이 확인하세요

PC Chrome에서 Companion을 설치하면 내 Instagram 프로필의 팔로워와 팔로잉 목록을 화면에서 수집하고 맞팔 아닌 명단을 계산합니다. 기존 공식 ZIP 분석도 그대로 사용할 수 있습니다.

현재 관계 스캔공식 ZIP 분석

1. Companion 설치

  1. 웹의 Companion 받기를 누릅니다matchal-companion-v23.zip 파일을 내려받습니다.
  2. 압축을 폴더에 풉니다Chrome 확장 프로그램 관리 화면에서 개발자 모드를 켜고 풀린 폴더를 불러옵니다.
  3. 맞팔체커를 새로고침합니다Companion 연결됨이 표시되면 준비가 끝납니다.

2. 현재 관계 스캔

  1. 같은 Chrome에서 Instagram에 로그인합니다내 프로필을 열고 Companion 사이드패널을 실행합니다.
  2. 내 아이디를 감지하거나 입력합니다로그인한 계정의 내 프로필만 스캔할 수 있습니다.
  3. 팔로워·팔로잉 스캔 시작을 누릅니다두 목록을 차례로 열어 화면에 로드되는 아이디를 수집합니다. 완료될 때까지 Instagram 탭을 닫지 마세요.

팔로워 명단

현재 나를 팔로우하는 계정입니다. 아이디 검색, 프로필 열기와 CSV 저장을 지원합니다.

팔로잉 명단

현재 내가 팔로우하는 계정입니다. Instagram 화면에서 수집된 시점의 명단입니다.

맞팔 아닌 명단

팔로잉에서 팔로워를 뺀 결과입니다. 나는 팔로우하지만 상대는 현재 나를 팔로우하지 않는 계정입니다.

목록이 일부일 수 있는 경우

목록 로딩이 멈추거나 탭을 닫으면 일부 명단으로 저장될 수 있습니다. 경고가 표시되면 다시 스캔하거나 공식 ZIP 분석을 이용하세요.

3. 맞팔 아닌 명단 활용

  1. 명단을 먼저 검토합니다지인, 브랜드, 업무 계정처럼 유지할 대상은 제외하세요.
  2. 필요한 계정만 작업 목록으로 보냅니다웹에서는 개별 선택이 가능하고 Companion에서는 전체 맞팔 아닌 명단을 작업 목록으로 만들 수 있습니다.
  3. 처음에는 2~3개만 시험합니다처리 수, 대기 시간과 휴식 설정을 확인한 뒤 실행합니다.

4. 과거에 나를 언팔한 계정 찾기

현재 스캔 한 번으로는 과거의 언팔 기록을 알 수 없습니다. Instagram 계정 센터에서 팔로워 및 팔로잉을 JSON 형식으로 서로 다른 날짜에 내려받고, 맞팔체커의 관계 변화 비교에 이전 ZIP과 최신 ZIP을 넣으세요.

5. 처리 수와 안전 간격

설정기본값·범위의미
이번 처리 수기본 10개 · 최대 30개한 번에 처리할 최대 계정 수
계정 사이 대기기본 8~15초각 프로필 사이의 임의 대기 시간
정기 휴식기본 10건마다 60초연속 작업 사이의 긴 휴식
안전 중지로그인·보안 확인·연속 오류문제가 감지되면 작업 중지
