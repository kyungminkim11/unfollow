# 맞팔체커 Companion v22

맞팔체커 웹에서 만든 `팔로워 이탈 후에도 내가 팔로우 중인 계정` 또는 `현재 비맞팔 계정` 목록을 Chrome 사이드패널로 받아 Instagram 프로필 화면의 팔로우 취소 버튼을 순서대로 처리합니다.

## 설치

1. 배포 사이트에서 `matchal-companion-v22.zip`을 내려받습니다.
2. ZIP을 폴더에 압축 해제합니다.
3. Chrome 주소창에 `chrome://extensions`를 입력합니다.
4. 오른쪽 위 `개발자 모드`를 켭니다.
5. `압축해제된 확장 프로그램을 로드합니다`를 눌러 이 폴더를 선택합니다.

## 동작 방식

- 맞팔체커 웹이 아이디 목록만 확장 프로그램 로컬 저장소로 전달합니다.
- 사용자가 사이드패널에서 처리 수, 대기 시간, 휴식 시간을 설정하고 확인란을 선택한 뒤 시작합니다.
- 확장 프로그램은 Instagram 프로필을 한 명씩 열고 화면에 표시된 `팔로잉`과 `팔로우 취소` 버튼을 찾습니다.
- 로그인 화면, 보안 확인 화면, 연속 오류가 감지되면 중지합니다.
- 비밀번호, 인증 코드, 쿠키, 세션 토큰 또는 Instagram 비공개 API를 읽거나 호출하지 않습니다.

## 기본 보호 설정

- 한 번에 최대 30개
- 계정 사이 최소 5초 이상
- 기본 8~15초 임의 대기
- 기본 10건마다 60초 휴식
- 일시정지와 즉시 중지
- 2회 연속 DOM 탐지 실패 시 자동 중지

Instagram 화면 구조나 표시 언어가 바뀌면 버튼 탐지가 실패할 수 있습니다. 실패 항목은 자동 재시도하지 않고 사이드패널에서 확인 후 다시 대기 상태로 변경합니다.
